Background selection

One can use GUI to interactively select time range as background data, or use the method LightCurve.background_range_list, example:

[10]:

T0 = lc.trigger_time
lc.background_range_list = [[T0, T0+50], [T0+200, T0+250]]
lc.plot(color='blue', GUI=True)
